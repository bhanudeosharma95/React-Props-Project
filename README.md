# React Props Project

A simple React project created to understand and practice the concept of **Props** in React. This project demonstrates how data can be passed from parent components to child components, making components reusable and dynamic.

## 🚀 Features

- Passing data using props
- Reusable React components
- Dynamic content rendering
- Clean and simple UI
- Beginner-friendly project structure

## 🛠️ Technologies Used

- React
- JavaScript
- HTML
- CSS

## 📂 Project Structure

```bash
src/
├── components/
│   └── Card.jsx
├── App.js
├── index.js
└── ...
```

## ▶️ Getting Started

### Clone the repository

```bash
git clone https://github.com/bhanudeosharma95/React-Props-Project.git
```

### Navigate to the project folder

```bash
cd React-Props-Project
```

### Install dependencies

```bash
npm install
```

### Start the development server

```bash
npm start
```

The application will run at:

```bash
http://localhost:3000
```

## 📚 Learning Objective

The main goal of this project is to understand:

- What are Props in React?
- How to pass data from parent to child components
- How to create reusable components
- Component-based architecture in React
